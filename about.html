About

What is Strains?

Strains is a site where you can find more information about a cannabis strain. We have different categories where you can explore different strains. You can search by strain type (Sativa, Indica, Hybrid), flavor, and mood effects!

This site was created for a school project. We created this site because we were introduced to API's in our class, "Emerging Web Tools and Trends".

We used used The Strain API to create this site.
